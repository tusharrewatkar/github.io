Neon Cube - A Puzzle Platformer Game

Neon Cube is a vibrant puzzle-platformer mobile game developed in Godot. It combines neon-themed visuals with Geometry Dash styled progression, available on Android, iOS, and PC.

Neon Cube is available on:

  • Android and iOS: Mobile devices with touch controls.
  • PC: Keyboard and gamepad support.

Media and Gameplay

The game was developed using:

  • Godot: Game engine used for scripting, level design, and physics.
  • SmartShape2D: Library for creating neon-style graphics and smooth shapes.
  • Aseprite: Software for creating sprite assets and animations.
  • Firebase: Backend service for leaderboards and user score tracking.
  • Admob: Platform for integrating ads on mobile platforms.
